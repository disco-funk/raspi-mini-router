# raspi-mini-router
A tiny little mini bash script that changes a Raspberry Pi into a dinky little NAT router


## Installation

From project root, run

```bash
sudo bash install.sh
```

Router kickoff script will be installed as `router` in the folder `/usr/local/sbin/`. Generally on a OOTB Raspberry Pi, this folder will be on your `PATH` already.

## Run

To kick off the router:

```bash
router
```
